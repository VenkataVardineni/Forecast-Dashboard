# Forecast Dashboard

A dashboard application for forecasting and data visualization.

## Getting Started

### Prerequisites

- Python 3.8+
- Node.js 16+ (if using frontend)

### Installation

```bash
# Install Python dependencies
pip install -r requirements.txt

# Install Node dependencies (if applicable)
npm install
```

### Usage

```bash
# Run the application
python app.py
```
